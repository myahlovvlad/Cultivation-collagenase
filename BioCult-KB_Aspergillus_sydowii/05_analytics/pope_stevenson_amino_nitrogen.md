# Метод Попе–Стивенсона

Определяет аминный азот по образованию комплексов аминокислот и пептидов с медью и последующему йодометрическому титрованию.

Формула:

`X = (a − b) · 0.28 · K · 100`

где `a` — объём гипосульфита для пробы, `b` — объём для контроля, `K` — поправочный коэффициент титра.

Статус: `FACT`.
