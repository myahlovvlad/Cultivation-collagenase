# Аналитические контроли

## Минимальный набор

1. Коллаген + буфер без образца.
2. Образец + буфер без коллагена.
3. Кипячёный образец + коллаген.
4. Образец + коллаген без ингибитора.
5. Реактивный бланк нингидрина.

## Исправленная КЛА

`КЛА_corr = валовый сигнал − фон образца − неферментативный фон − реактивный фон`.

## Ингибиторы

- PMSF — сериновые протеазы;
- EDTA — металлозависимый вклад;
- 1,10-фенантролин — металлозависимый вклад;
- комбинации — смешанный пул.

## Субстраты для A/B

Нативный коллаген I, денатурированный коллаген, желатин, казеин.
