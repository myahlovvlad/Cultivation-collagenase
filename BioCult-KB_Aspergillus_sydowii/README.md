# BioCult-KB_Aspergillus_sydowii

Единая база знаний для темы: **«Создание прототипа биоаналитической экспертной системы для прогнозирования выхода коллагенолитических ферментов при культивировании *Aspergillus sydowii* ВКПМ F-347 в лабораторном биореакторе».**

## Принцип единой системы правды

1. Первичные документы проекта имеют приоритет над диалоговыми интерпретациями.
2. Все значения маркируются статусом: `FACT`, `CALCULATED`, `INTERPRETED`, `ASSUMPTION`, `HYPOTHESIS`, `CONFLICT`, `TODO_VERIFY`.
3. Конфликты не удаляются: они вынесены в `00_project_passport/unresolved_issues.md`.
4. Текущая рабочая среда v1: **меласса 20 г/л + пептон 85 г/л**.
5. Экспериментальный кандидат оптимума: **меласса 30 г/л + пептон 85 г/л, ΣКЛА 136.43 ЕД/мл**.
6. До ингибиторного профилирования используется термин **«внеклеточный коллагенолитический ферментный пул»**, а не строго доказанная индивидуальная коллагеназа.

## Структура

`00_project_passport` — рамка проекта;  
`01_strain` — штамм;  
`02_media` — питательные среды;  
`03_cultivation_process` — культивирование;  
`04_bioreactor` — биореактор;  
`05_analytics` — аналитика;  
`06_experimental_data` — данные;  
`07_models` — модели;  
`08_expert_system` — правила;  
`09_reports_and_outputs` — итоговые документы.

## Прототип Web-приложения

В корне добавлен каталог `web_app` с прототипом web-приложения на FastAPI и SQLite. Оно позволяет:

- просматривать справочники штаммов и сред
- импортировать компоненты среды из Excel
- рассчитывать ключевые метрики по наблюдениям
- генерировать рекомендации на основе правил

Запуск: `cd web_app && python -m venv .venv && .\.venv\Scripts\activate && pip install -r requirements.txt && uvicorn main:app --reload`.
