# Среда Чапека-Докса с коллагеном

Использовалась на раннем этапе для проверки способности микромицетов продуцировать коллагенолитическую активность в присутствии коллагена.

## Трактовка

Это скрининговая среда, а не текущая продукционная среда. Она нужна для обоснования выбора *A. sydowii* как более перспективного продуцента.

## Статус

`FACT + INTERPRETED`.
