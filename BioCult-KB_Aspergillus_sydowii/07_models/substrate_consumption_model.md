# Модель потребления субстратов

Переменные:

- `S(t)` — сахара, мг/мл;
- `N_amino(t)` — аминный азот, мг%;
- `Protein(t)` — белок, мг/мл.

Назначение — оценить, связано ли изменение КЛА с истощением углерода, изменением органического азота или фазой роста.

Гипотеза: рост КЛА после снижения сахаров может указывать на частично не рост-ассоциированную продукцию ферментного пула.

Статус: `MODEL_V0`.
