# План отбора проб

## Базовые точки

0, 24, 48, 72, 96, 120, 144, 168, 192 ч.

## Обязательные показатели

- pH;
- pO₂;
- температура;
- rpm;
- пена;
- морфология;
- сухая биомасса;
- сахара;
- белок;
- аминный азот;
- КЛА при 570 нм;
- КЛА при 440 нм;
- ΣКЛА;
- удельная активность.

## Требование

Каждая проба должна иметь `sample_id`, `batch_id`, время отбора, объём, оператора и статус пригодности.
