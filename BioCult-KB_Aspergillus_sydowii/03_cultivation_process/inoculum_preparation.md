# Подготовка инокулята

## Зафиксированное значение

Объём инокулята для биореактора — 100 мл.

## Что нужно стандартизировать

- возраст инокулята;
- номер пассажа;
- среда для инокулята;
- способ стандартизации посевного материала;
- критерии пригодности;
- морфология перед внесением.

## Правило

Если культура после восстановления прошла менее 2–3 пассажей, запуск не использовать как опорный для параметров продуктивности.
