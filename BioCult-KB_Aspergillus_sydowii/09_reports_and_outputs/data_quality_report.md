# Качество данных

## Сильные стороны

Есть паспорт штамма, составы сред, таблицы КЛА, белка, сахаров, аминного азота, биомассы, данные по оптимизации среды и информация о биореакторе.

## Ограничения

- конфликт 20/85 и 30/85;
- нет полного pO₂-профиля;
- нет расхода воздуха;
- нет ингибиторного профиля;
- нет системного морфологического скоринга;
- геномный accession требует проверки;
- производитель фактического биореактора требует уточнения.

## Приоритет

Сначала стандартизировать биореакторный batch-run template и поставить A/B-тест среды.
