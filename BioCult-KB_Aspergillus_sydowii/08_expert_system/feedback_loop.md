# Обратная связь

Цикл:

1. Система задаёт программу культивирования.
2. Лаборатория проводит запуск.
3. Данные заносятся в шаблон.
4. Система сравнивает факт с ожидаемой траекторией.
5. Отклонения классифицируются.
6. Обновляются правила и модели.
7. Формируется новая рекомендация.

Ключевые KPI: ошибка прогноза ΣКЛА, повторяемость максимума, полнота контролей, количество отклонений.
