# Дерево решений

```text
Запуск batch
│
├─ pH 7.1–7.5?
│   ├─ да → продолжить
│   └─ нет → отметить отклонение
│
├─ pO₂ стабилен?
│   ├─ да → продолжить
│   └─ нет → проверить аэрацию/rpm/пену/морфологию
│
├─ t < 96 ч?
│   ├─ да → фокус на росте
│   └─ нет → фокус на продукции КЛА
│
├─ t = 144 ч?
│   ├─ да → полный аналитический контроль
│   └─ нет → продолжить
│
└─ КЛА снижается после 144 ч?
    ├─ да → harvest candidate = 144 ч
    └─ нет → уточнить модель
```
