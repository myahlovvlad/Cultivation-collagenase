# Цель и задачи

## Цель

Создать прототип экспертной системы, которая систематизирует данные и прогнозирует выход коллагенолитических ферментов при культивировании *Aspergillus sydowii* ВКПМ F-347.

## Задачи

1. Сформировать реестр источников.
2. Нормализовать сведения о штамме, средах, биореакторе и аналитике.
3. Сформировать мастер-таблицы данных.
4. Описать программу периодического культивирования.
5. Построить модели роста, продукции фермента, потребления субстратов, кислородного лимитирования и морфологии.
6. Сформировать правила экспертной интерпретации.
7. Задать контур экспериментальной обратной связи.

## Workflow

`источник → факт → нормализованное значение → модель → правило → рекомендация → новый эксперимент → корректировка`.
