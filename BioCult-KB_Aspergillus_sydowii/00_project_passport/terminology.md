# Терминология

**КЛА** — коллагенолитическая активность, измеряемая по накоплению свободных α-аминогрупп после гидролиза коллагена.

**ΣКЛА** — рабочий расчётный показатель, сумма активностей по каналам 570 и 440 нм. Используется как внутренний показатель проекта и требует явного методического обоснования.

**УКА** — удельная активность, отношение КЛА к концентрации белка.

**Коллагенолитический ферментный пул** — корректный термин до ингибиторного профилирования.

**Soft sensor** — расчётный виртуальный датчик, прогнозирующий КЛА по косвенным параметрам.
