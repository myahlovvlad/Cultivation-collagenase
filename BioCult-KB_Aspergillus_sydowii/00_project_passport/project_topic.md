# Тема проекта

**Создание прототипа биоаналитической экспертной системы для прогнозирования выхода коллагенолитических ферментов при культивировании *Aspergillus sydowii* ВКПМ F-347 в лабораторном биореакторе.**

## Смысл

Проект переводит разрозненные лабораторные данные в систему поддержки биотехнологического решения. Система должна связывать штамм, среды, параметры культивирования, биореактор, аналитику, модели и обратную связь.

## Границы

Текущая версия — не промышленный цифровой двойник, а **прототип экспертной системы** и база знаний для последующего цифрового двойника.
